# Python Calculator

A simple console-based calculator built using Python. This project demonstrates basic programming concepts such as user input, conditional statements, and functions.

## 🔧 Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly console interaction

## 🚀 How to Run

```bash
git clone https://github.com/CodeCrafter-1236/python-calculator.git
cd python-calculator
python calculator.py
```

## 🧠 Concepts Used
- Python basics
- Conditional statements (if-else)
- Functions
- User input handling

## 📌 Sample Output

```
Enter first number: 10
Enter operator (+, -, *, /): +
Enter second number: 5
Result: 15
```
